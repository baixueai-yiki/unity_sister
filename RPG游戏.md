# 项目目标
制作一个完整的小游戏，游戏内容简单一些就好
关键是包含完整的游戏流程：外部菜单、关卡过程、游戏结算
现在的思路是制作一个极简的中世纪风格RPG游戏

再之后，据说需要做一些商业项目，来展示技能深度
我不太清楚这包括什么


# 开发流程
1.玩家单位与地图设计
把村庄的地图放进去，做完玩家单位再视情况调节大小
设计玩家单位，摄像机始终跟随玩家
鼠标右键点击地图后单位平移过去，并根据状态播放不同贴图
2.填充npc与对话系统
在地图里放一些不会动的npc，左键点击对话
设计对话框，调用文本内容
3.填充敌人与战斗系统
在地图里放一些会动的敌人，左键点击攻击



# 目录结构
Assets/
├── Scenes/         # 场景（Unity默认）
│   ├── Main/           # 主流程（主菜单、主关卡）
│   ├── Levels/         # 各关卡
│   └── Test/           # 测试用场景
│
├── Scripts/        # 代码
│   ├── Core/           # 核心系统（GameManager等）
│   ├── Player/         # 玩家相关脚本
│   ├── Enemy/          # 敌人相关脚本
│   └── UI/             # UI逻辑脚本
│
├── Art/            # 美术资源
│   ├── Models/         # 模型
│   ├── Textures/       # 贴图
│   ├── Materials/      # 材质
│   └── Animations/     # 动画
│
├── Audio/          # 音频
│   ├── BGM/            # 背景音乐
│   └── SFX/            # 音效
│
├── Prefabs/        # 预制体
│   ├── Player/         # 玩家预制体
│   ├── Enemy/          # 敌人预制体
│   └── UI/             # UI预制体
│
├── UIResources/    # UI资源（图片、字体等）
│
├── Data/           # 数据（可选，但建议加上，用于配置表/存档等）
│
├── Plugins/        # 第三方插件
└── Settings/       # 系统配置（尽量不要乱动）
